# Monthly-sales-MIS-report
Created an interactive Monthly Sales MIS Dashboard using Microsoft Excel to analyze sales and profit performance across products and cities.

# Tools & Features Used:
Pivot Tables
Pivot Charts
Slicers
KPI Cards
Conditional Formatting
Power Query
Excel Formulas

# Key Highlights:
Analyzed monthly sales and profit trends
Built dynamic KPI cards for Total Sales, Profit, Avg Sales, and Highest Sales
Created interactive slicers for City and Product level analysis
Designed visual dashboards using charts and Pivot Tables
Generated business insights and recommendations for performance improvement

# Key Insights:
Bangalore recorded the highest sales and profit performance
Laptop was the top-performing product
Chennai showed the lowest sales contribution
Identified opportunities to improve low-performing products and cities

# Business Recommendations:
Focus on high-demand products
Improve marketing strategies in low-performing cities
Optimize pricing and promotional offers
Collect customer feedback to improve sales performance

# Outcome:
Developed a professional MIS reporting dashboard to support business decision-making and performance analysis.
